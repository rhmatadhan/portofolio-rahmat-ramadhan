# Portfolio - Rahmat Ramadhan

Website portofolio statis dengan Tailwind CSS (CDN). Fokus pada tampilan modern, clean, dan responsif.

## Preview
Buka file [index.html](index.html) langsung di browser.

## Struktur
- assets/css/ style tambahan
- assets/js/ interaksi sederhana (navbar, reveal)
- assets/images/ gambar dan screenshot
- index.html halaman utama

## Catatan
- Semua link demo dan sosial bisa diganti langsung di index.html.
- Untuk host, bisa gunakan GitHub Pages atau layanan static hosting lainnya.
